Pomodoro-Timer

Der Begriff "Pomodoro" kommt aus dem Zeitmanagement: 25min Arbeitsintervalle und 5min Pausenintervalle wechseln einander ab. Mein Pomodoro-Timer ist ein Tool, bei dem Arbeits- und Pausenzeiten frei eingestellt werden können. Er is praktisch, und sieht noch nebenbei zum Anbeissen aus.

Mein Pomodoro-Timer verwendet das populäre Bootstrap-Framework in Kombination mit jQuery. Herzstück des Javascript-Codes ist die Funktion setInterval(update_disp, 1000) als Taktgeber. Sobald der Start-Button gedrückt wird, wird im Sekundentakt update_disp() aufgerufen. Dadurch werden der Countdown und der Fortschrittsbalken aktualisiert. Mittels trigger_break() wird automatisch zwischen Arbeits- und Pausenintervallen gewechselt. Durch das Drücken des Stop-Buttons wird der Timer via clearInterval() angehalten. Mehr zu meinem Code auf Github .

Random Quotes

Random Quotes zeigt zufällig ein Zitat einer berühmten Persönlichkeit an, das außerdem bei Twitter gepostet werden kann.

15 vordefinierte Zitate befinden sich in einem 2D-Array, der jederzeit erweitert werden kann. Via Javascript wird ein Zitat zufällig ausgewählt und via jQuery angezeigt. Weiters können Zitate bei Twitter gepostet werden. Mehr zu meinem Code auf Github .
